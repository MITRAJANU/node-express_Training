# JavaScript Refresher For Node, Express, And React

Run all examples:

```bash
npm run js:refresher
```

Run one example:

```bash
node examples/js-refresher/03-arrays-map-filter-find.js
```

## Teaching Order

| File | Concept | Where students use it |
|---|---|---|
| `01-let-const-scope.js` | Scope, `let`, `const` | Every JS project |
| `02-functions-and-closures.js` | Functions, callbacks, closures | Express middleware, React handlers |
| `03-arrays-map-filter-find.js` | Array methods | Task filtering, React list rendering |
| `04-objects-destructuring-spread.js` | Objects, destructuring, spread | `req.body`, response shaping, React state |
| `05-modules/app.js` | ES modules | Modern Node, Express routes/controllers, React imports |
| `06-promises.js` | Promises | DB calls, API calls |
| `07-async-await-try-catch.js` | async/await, try/catch | Express controllers, React API handlers |
| `08-promise-all.js` | Parallel async work | Loading independent data together |
| `09-json-and-fetch.js` | JSON, fetch | REST APIs and frontend API calls |
| `10-error-handling.js` | Error objects, custom errors | Express error middleware |
| `11-react-state-patterns.js` | Immutability | React state updates |
| `12-node-express-patterns.js` | Request object patterns | Express routes/controllers |
